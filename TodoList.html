To Do List

Project Overview

This project as the title suggests is a To do List Website using Html, CSS and Javascript. There are two buttons that allow the user to add and delete tasks. Each task is numbered and has a checkbox indicating that the task is done. This was my first completed project on using Front End development languages and this website is hosted on my github page.
