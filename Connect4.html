Connect 4

Project Overview

This project is a recreation of the game Connect 4 using Python The basic structure of the game has you and one other person taking turns manually typing numbers in order to place spots in a grid. Like Connect 4 in real life, the first person to create a line for 4 points wins.

My Process

The basic outline for tackling this problem was taking the Connect 4 game and breaking it into smaller parts. The first part was making a grid composed of rows and columns to simulate a Connect 4 game board. Each column would have a corresponding letter used to indicate where to place a players game piece.

The second part was having the program react to the player’s inputs (in the form of letters) and placing the player’s symbol at the bottom most point of the specified column. This is done to simulate the act of dropping chips into the grid in real life Connect 4.

Lastly, I had the program update the grid and then detect if either player had gotten a line four. If the program found one they would end the program and declare the winner.
